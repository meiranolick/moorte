History:

  • Element begins with the Tango skin.
  • Detach - Disconnects the element from the rte and turns off editing. The toolbar and other elements connected to the toolbar are unaffected.
    Restore reattaches the element to the rte it was last connected to, unless a different RTE is passed in.
  • Hide - Adds "display:none" to the rte toolbar. Simple hide, nothing more.
    Restore RTE removes the display:none, and everything is back to where it was.
  • Remove - removes the rte toolbar from the page and makes all connected elements uneditable.
    Restore RTE reverses the actions, and all is back to how it was.
  • Destroy - destroys the rte toolbar and makes all connected elements uneditable.
    Restore creates a brand new RTE, with the default Word'03 skin.

Geeks Only:
In concept, "detach" is a function of element. It detaches the element from the RTE, without affecting the RTE at all.
Detach therefore can NOT be called directly on a RTE.

Restore, Remove and Hide are methods of the RTE: They affect the RTE and every element associated with it.
For practicality, restore/remove/hide can be called on any element which has an associated RTE, the RTE is assumed.
